# 데이터 베이스
: 데이터를 효율적으로 저장하고 관리하기 위한 시스템

# 종류
: Oracle, MySQL, MS-SQL(SQL Server)

# SQL (Structured Query Language)
: 구조화된 질의어 - "데이터베이스 언어"

# MySQL
1. MySQL 설치
2. 데이터베이스 계정 생성 및 접속
3. SQL
    - DDL : 데이터 정의어 - 객체(구조) 정의
    - DML : 데이터 조작어 - 데이터 조회/추가/수정/삭제
    - DCL : 데이터 제어어 - 계정 권한 부여/해제
    - TCL : 트랜잭션 제어어 - 변경 사항 적용/복원
